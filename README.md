# Listenabstraktion_fuer_Wochentage
Dieses Python-Skript demonstriert die Effizienz von Listenabstraktionen (List Comprehensions). Aus einer Liste aller Wochentage wird eine neue Liste generiert, die nur jene Tage enthält, deren Namen mit dem Buchstaben "S" beginnen. Dies ist ein kompaktes Beispiel für die Filterung von Daten in einer Sequenz.
